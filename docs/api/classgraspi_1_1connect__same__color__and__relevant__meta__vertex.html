class graspi::connect_same_color_and_relevant_meta_vertex¶

This is predicate to filter the graph that connects vertices to the relevant meta-vertex

Predicate used to filter original graph to find the shortest distances from source vertex - the meta-vertex of relevance and any other vertices where edges between vertices exist only if they connect vertices of the same color or are connected to top or bottom meta-vertices.
